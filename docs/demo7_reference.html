Demo 7 Canvas2D Reference — dirty rect 差分更新の「見た目の正解」

t = 0.0 s
このページは jsengine の Demo 7 と同じ描画コードをブラウザの Canvas 2D API で実行します。
  • サイズは 640×480、毎フレーム時刻 time ms に依存して動く黄色矩形
  • 背景（青→赤グラデーション + グリッド）は 初回フレームだけ 描画されるのを jsengine 側と合わせるため、ブラウザ側も demo7Inited フラグで同じ挙動にしている
  • dirty rect 最適化自体は jsengine 内部の最適化なので、ブラウザでは「見た目の正解」を確認するのが目的
jsengine 側との差異 (期待される):
  • dirty rect は jsengine 内部の GL テクスチャ部分アップロード機構。ブラウザは関知しない（全画面 redraw で代用）
  • 「前フレーム位置をクリア → 背景再描画 → 新位置に描画」の流れ自体は両方で同じ
  • 背景再描画は 4px 単位の縞をクリア範囲だけに引き直すので、jsengine 側でズレがあれば「縞の継ぎ目」として見える
