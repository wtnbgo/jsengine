Demo 4 Canvas2D Reference — Canvas2D アニメーションの「正解」

t = 0.0 s
このページは jsengine の Demo 4 と同じ描画コードをブラウザの Canvas 2D API で実行します。
  • サイズは 512×512、毎フレーム時刻 time ms に依存して動く
  • jsengine 側との比較のため、一時停止と 1 フレーム進めるボタンを用意
  • 時刻は performance.now() ベース。jsengine 側は SDL_GetTicks ベース。両者を同じ t 値（秒）に揃えて並べると比較しやすい
jsengine 側との差異 (期待される):
  • FPS が違うので「同時刻でのコマ」を直接比較するには t 値の表示を見ながら一時停止する必要あり
  • 軌道上の小さな円は rgba(255,255,255,0.5) なので背景色 (#16213e) と混色して見える
