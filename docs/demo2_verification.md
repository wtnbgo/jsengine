## 描画内容（上から下、左から右）

### 上段 — 矩形と円（Y=30〜130 付近）

| 位置 | 内容 | 色 | 備考 |
|------|------|-----|------|
| (30,30) 120×80 | 塗り矩形 | `#e94560` (赤桃) | |
| (60,60) 120×80 | 半透明矩形 | `#16213e` α=0.5 | 赤桃の上に重ねる |
| (180,30) 120×80 | 枠線矩形 | `#0f3460` 線幅4 | 内側は背景色 |
| (400,80) r=50 円 | 紫の円 | `#533483` | |
| (400,80) r=50 半円 | 黄半円 | `rgba(255,200,0,0.7)` | 上半分のみ |

### 中段 — パス描画（Y=180〜320 付近）

- **三角形** (50,250)→(150,180)→(150,320): 緑塗り `#2b9348` + 白線幅2 で輪郭
- **ベジェ曲線** (200,250)→(450,250): 赤線 `#ff6b6b` 線幅3 で波形（コントロール点 (250,150),(350,350)）

### 下段 — 矩形パターン (Y=380〜430)

5 色の 50×50 矩形が x=30, 90, 150, 210, 270 に並ぶ。色順: 赤・青・緑・黄・橙。

### 右下 — 同心円 (中心 400,400)

半径 20→100 の 5 重円。外側ほど赤成分が強く、α=0.6 で重なり合う。

## チェックポイント

- [ ] **半透明合成**: 赤桃矩形の右下に半透明 `#16213e` が重なり、混色 (`#7d2f3e` 風) になる
- [ ] **円弧の真円度**: 紫の円が真円に見え、ジャギーが目立たない（GLES のラスタライズは ThorVG 経由でアンチエイリアス）
- [ ] **ベジェの形**: 中央が上下に膨らんだ S 字波形
- [ ] **同心円のα合成**: 中心付近が一番濃く、外周ほど薄い
- [ ] **線の太さ**: 三角形の白線は細く (lw=2)、ベジェの赤線は太い (lw=3)、外周矩形の青線は中 (lw=4)

## よくある不具合パターン

- 半透明矩形が完全不透明 → `globalAlpha` がリセットされていない
- 円が多角形に見える → ThorVG の `arc()` の分割数不足
- ベジェが直線になる → `bezierCurveTo` の制御点処理がバグ
- 同心円の色が均一 → α合成が正しく機能していない
