Demo 6 Canvas2D Reference — drawImage / getImageData / putImageData の「正解」

このページは jsengine の Demo 6 と同じ描画コードをブラウザの Canvas 2D API で実行します。
  • サイズは 500×700 固定。テスト用画像 data/test_pattern.png を読み込んで描画
  • jsengine 側は Image/HTMLCanvasElement シム経由で「ブラウザと同じ感覚」のフローも検証している
  • ブラウザ側はそれらが native なので、シム経路の B/C/D セクションは「ブラウザでは当然動く」前提
jsengine 側との差異 (期待される):
  • セクション 2 (Image シム経由): jsengine では testImg._data の存在やバイト数も表示される。ブラウザでは native HTMLImageElement なので _data は undefined 扱い、その代わり実画像が出る
  • セクション 3 (getImageData/putImageData): ブラウザ・jsengine 両方とも同じ色反転結果になるはず
