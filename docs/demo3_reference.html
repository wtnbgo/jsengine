このページは data/fonts/ の同じ TTF/OTF を @font-face で読み込み、 jsengine の Demo 3 と同じ描画コードを Canvas 2D API で実行します。 ローカル file:// で開いた場合、ブラウザによっては font 読み込みがブロックされます。 その場合は python -m http.server 等で D:/test/jsengine/ を配信し、 http://localhost:8000/docs/demo3_reference.html で開いてください。
jsengine 側との差異:
  • textLocale はブラウザ標準ではないので無視されます（FT loader 用の言語ヒント）。CJK 表示はブラウザ自身の font fallback に任せる形になります。
  • 絵文字: ブラウザはシステム標準のカラー絵文字を使う場合あり。jsengine は NotoEmoji-Regular のモノクロ字形を使うので塗りつぶしの形になります。
  • NotoEmoji-Regular だけを指定した行で、ブラウザがシステムフォントに fallback して立体絵文字を表示することがあります。これは jsengine 側の見え方とは異なるが、各文字が「出ている」こと自体は両方で確認できます。
