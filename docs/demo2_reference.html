Demo 2 Canvas2D Reference — Canvas2D 図形描画の「正解」

このページは jsengine の Demo 2 と同じ描画コードをブラウザの Canvas 2D API で実行します。 並べて見比べることで、矩形・パス・円・ベジェ・α合成のズレが分かります。
  • 静的描画（アニメーションなし）
  • サイズは 512×512 固定
  • jsengine 側は 3 キーで Demo 3 → 2 キーで Demo 2 に切替（実体は Demo 番号 2）
jsengine 側との差異 (期待される):
  • なし — 全ての形状・色・α合成・パスは Canvas2D 標準動作内で完結するため、見た目はピクセル単位で近いはず。
  • ベジェ曲線・円弧の補間アルゴリズム差で 1〜2px のエッジ差は出る可能性あり。
