Demo 10 pixi.ui Reference — pixi.ui ウィジェットの「正解」動作

click any widget
このページは jsengine の Demo 10 と同じ初期化コードをブラウザで実行します。 data/lib/pixi.min.js, pixi-ui-deps-shim.js, pixi-ui.js をそのまま読み込み。
  • サイズは 1280×720
  • Button: クリックで背景色がサイクル変更
  • CheckBox: クリックで Animate (ProgressBar) ON/OFF
  • Slider: ハンドルをドラッグで値変更 (0〜100)
  • ProgressBar: CheckBox ON 時のみ進む
  • ScrollBox: マウスホイールでスクロール
jsengine 側との比較ポイント:
  • ブラウザ側はクリック・ドラッグ・ホイールが native に動く → 全ウィジェットが期待通り反応する
  • jsengine 側でクリック反応がない場合、原因は browser_shim.js の pointer event 変換にある
  • tweedle はシム実装（即時遷移）なので、本来のアニメーションが見たい場合は CDN 版 tweedle を使う必要あり（このページでも同じシム）
